用 Start 云游戏在 Mac 上玩国服《英雄联盟》

2020-10-27 · 5 min read
Game Life

这是一篇发布于 8 个月前的文章，其中的信息可能已经有所发展或是发生改变

虽然现在 5G 在消费端还没带来什么变化，倒是给我们带来了几个新概念，「云游戏」算是其中一项。无论是拥有自己游戏工作室的 Microsoft、驻足社交领域的 Facebook、还是搜索引擎龙头的 Google，都在发展各自的云游戏。而作为国内游戏市场「顶梁柱」的腾讯（虽然的确不喜欢），也给国内目前尚未得到耕耘的「云游戏」市场，带来了自己的项目 —— Start 云游戏

Mac 游戏的现况

要说 Mac 用户在聊到什么使用场景会尝试「退出群聊」，那无疑就是游戏场景了。实话实说，Mac 上的游戏资源是真的少。除了 Mac App Store 上的游戏外，Steam 等第三方游戏平台中的游戏对 Mac 的支持可谓是「少之又少」

作为 Mac 单平台用户，Windows 最让我羡慕的便是其丰富的游戏支持

今年的 WWDC，Apple 宣布预计将用 4 年左右的时间将 Mac 过渡到 Apple Silicon，即自研 ARM 架构处理器上。这一过渡所带来的其中一个潜在的好处，就是可以极大丰富 Mac 上目前的游戏资源

腾讯 Start 云游戏体验

何谓「云游戏」

或许有必要先解释一下，何谓「云游戏」

传统模式下，所有游戏的运行基于本地设备的硬件配置，比如 Switch 游戏依托于本地的 Switch 硬件（不考虑破解、模拟等特殊情况）、PC 游戏依托于 PC 的硬件配置等。想要在一台老旧的 PC 上运行一款游戏大作，可能就需要为其升级硬件配置已满足游戏运行所需

传统模式下，游戏的运行依托于合适的本地设备硬件配置

而云游戏将可以让用户只使用网络，不依赖于本地设备的硬件条件，便可直接畅玩游戏。只要你的网络条件达到要求，便可以轻松做到这一点。云游戏有些类似于将我们的手机、电脑等变成一台显示器，连接上云端的电脑，便可畅玩其中的游戏

有了云游戏，在手机上畅玩游戏大作也不再是梦想

5G 目前在消费端看起来也就是网络连接速度快了些、延迟低了些，而这些新的特性正好与云游戏的要求相契合。当然，这并不是说只有 5G 才能做到这些

体验 Start 云游戏

从此 Mac 也有庞大游戏库

想要使用腾讯 Start 云游戏，你需要前往 Start 云游戏 官网 下载对应客户端。目前，Mac 和 Windows 平台正在进行公测（Windows 要先申请「不限量公测资格」，移动版和 TV 版目前仍在开发中

移动版和 TV 版目前仍在开发中

Start 云游戏目前仅支持使用 QQ 账号登录，登录后便可直接开始游戏了。目前已上线的支持游戏包括《剑灵》、《流放之路》、《QQ 炫舞》、《极品飞车 ONLINE》、《战争雷霆》、《NBA2K Online 2》、《穿越火线》、《英雄联盟》、《FIFA Online 4》、《枪神纪》、《QQ 飞车》、《逆战》和《堡垒之夜》

目前提供的部分云游戏

由于云游戏只依赖于稳定的网络连接，上述几款游戏在我体验的过程中也未让我的 MacBook Pro 出现散热问题（参考机型：2015 年初配备 13.3 英寸 Retina 显示屏的 MacBook Pro，内存为 8 GB），总体来说表现优异

个人的运行建议如下：

  • 尽量使用有线网络连接而非无线连接​。如有必要，请连接 5GHz 无线网络
  • 在使用过程中可以关闭蓝牙，以确保蓝牙不会影响到无线网络连接
  • 在使用过程中尽量使用交流电源​

无论是鼠标点击还是键盘操作，Start 云游戏都能给你带来堪比 Windows PC 的使用体验。需要注意的是，由于 Mac 的分辨率与 Windows PC 的分辨率一般存在一定的区别，在运行云游戏时，设备的显示分辨率会被自动调整。如果感到自动调整后的分辨率让你有些许不舒服感，可自行前往游戏内的显示设置进行进一步的调整

不出意料，Start 云游戏会在正式上线时采取和目前游戏加速器类似的付费政策。不说了，抓紧每一分每一秒免费白嫖的时间～
