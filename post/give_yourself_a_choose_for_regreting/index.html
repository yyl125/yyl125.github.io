尝鲜有风险，测试需谨慎

内容更新 本文内容将随状况的发展而更新

Apple 如很多操作系统开发商一样，除了有用于正式环境的软件版本外，还有专供开发者做新功能适配和 App 机型测试的 Developer Beta（开发者测试版本，下文简称「DB」）以及面向普通消费者用户的 Public Beta（公众测试版本，下文简称「PB」）。每年的 WWDC（Apple 全球开发者大会）第一天的 Keynote 主旨演讲后，Apple 就会在其开发者网站上上线用户开发者的 DB 描述文件（Profile）用以开发者将自己的设备更新至新版本以为自己的 App 适配新系统做好准备

近些年来，随着相关门槛的降低以及互联网探索精神，越来越多的人开始尝试在自己的设备上尝试使用最新的测试版本来体验新功能。这里也要说明一下，测试系统仍然有很多不太完善的地方，因此其仅应面向有一定技术实力的开发者推出（虽然也有很多方式能够让公众获得）。而测试系统之所以被称之为测试系统，一定是因为它的完善度等方面还远没有达到可以面向一般大众推出的水平，不然早就全量推送了。如果无脑的尝鲜的话，遇到问题时也往往会让你付出一定的代价。这个代价包括但不限于：你的时间成本和你的资料数据等内容

也正因如此，如果你的的确确认为你自己有一定的技术实力而可以进行尝鲜的话，也不妨试着先留给自己一粒「后悔药」，以免后续遇到什么没法解决的问题时还能够尝试恢复先前的状态

再次提醒大家：测试有风险，尝鲜需谨慎

本文将依次介绍 iOS/iPadOS 以及 watchOS 的备份与还原办法。由于 Apple TV 购买项目和数据均储存在云端，因而无需备份你的 Apple TV

iOS/iPadOS

之所以将这两个系统放在一起说，是因为它俩虽然名字不一样，但在备份与还原方面的内容基本是一致的

备份和恢复方式的选择

备份 iPhone 或 iPad 的方法主要有两种，即通过 iCloud 和使用 PC/Mac。这二者在一些地方有些不同，需要注意一下：

  • iCloud 会将备份存储于 iCloud 云端；而使用 PC/Mac 备份则会讲备份存储于 PC/Mac；

  • iCloud 的储存空间随 Apple ID 是否订阅了 iCloud 空间而不同，最多提供 2 TB 的储存空间（最初 5 GB 免费）；而使用 PC/Mac 备份的储存空间则完全取决于你所用 PC/Mac 上的可用空间；

  • iCloud 备份将始终加密，这会保证你的「健康」、「健身记录」和「钥匙串」数据不会在恢复后丢失；而使用 PC/Mac 备份时，默认情况下不会加密，这将导致上述 3 个数据在恢复后丢失（因为不加密备份即不会备份上述 3 个数据）；

  • 在 iCloud 的备份几乎包括储存在你设备上的所有数据和设置，但仍然不包括：

    • 已储存在 iCloud 中的数据[1]；
    • 已储存在其它云服务中的数据；
    • Apple「邮件」数据；
    • Apple Pay 信息和设置；
    • Face ID 或 Touch ID 设置；
    • iCloud 音乐资料库和 App Store 内容；

  • 在 PC/Mac 上对设备进行备份包含设备的几乎所有数据和设置，但仍然不包括：

    • 来自 iTunes Store 和 App Store 的内容，或直接下载到 Apple Books 的 PDF；
    • 从「Finder 访达」或 iTunes 同步而来的内容，例如导入的 MP3 或 CD、视频、图书和照片；
    • 已存储在 iCloud 中的数据，如 iCloud 照片，iMessages 信息，以及文本（SMS，俗称「短信」）和多媒体（MMS，俗称「彩信」） 信息；
    • 面容 ID 或触控 ID 设置；
    • Apple Pay 信息和设置；
    • Apple「邮件」数据；
    • 「健身记录」、「健康」和「钥匙串」数据（需要开启加密备份）

你可以将一种设备的备份恢复到另一种设备（例如将 iPad 备份用于恢复 iPhone），但备份中某些类型的数据将无法传输到另一种设备上，包括照片、「信息」、「信息」中的附件、「语音备忘录」以及与你要设置的设备不兼容 App[2]

如果你选择通过 iCloud 备份和恢复

如果你选择通过 iCloud 备份，那么在恢复时你也只能从 iCloud 中恢复。具体的操作步骤如下：

  1. 将你的设备连接到 Wi-Fi 网络；
  2. 前往「设置」>「[你的姓名]」[3]，然后轻点「iCloud」；
  3. 轻点「iCloud 云备份」；
  4. 轻点「立即备份」。在备份过程结束前，保持你的设备始终处于 Wi-Fi 网络环境

按照上面的步骤操作，你的设备数据就会备份到你的 iCloud 中。此外，iCloud 云备份还支持每天自动备份，只需保证你的设备屏幕已锁定、连接到电源并处于 Wi-Fi 网络环境中即可[4]

那么，如何恢复呢？也很简单，跟着下面的步骤操作就好：

  1. 将要恢复的设备开机或先还原你的设备[5]，使其出现「您好」屏幕；
  2. 按照屏幕上所示的设置步骤操作，直到进入「App 与数据」屏幕，然后轻点「从 iCloud 云备份恢复」；
  3. 使用你的 Apple ID 登录 iCloud；
  4. 选取备份。查看各个备份的日期和大小，选择相关性最高的备份。选好备份后，传输便会开始。如果有信息提示需要更高版本的软件，请按照屏幕上所示的步骤进行更新[6]；
  5. 保持连接并等待进度条出现和完成。根据备份的大小和网络速度，可能需要数分钟到一个小时才能完成进度条。如果过早断开 Wi-Fi 连接，则进度条将暂停，直到重新连接 Wi-Fi；
  6. 继续完成设置并开始使用设备。App、照片、音乐和其它信息等内容将在接下来的数小时或数天内继续在后台恢复[7]，你需要经常连接到 Wi-Fi 并接通电源以完成这个过程

如果你选择通过 PC/Mac 备份和恢复

如果你选择通过 PC/Mac 备份，那么在恢复时你也只能从 PC/Mac 中恢复。具体的操作步骤如下：

  1. 将设备连接到电脑。在装有 macOS Catalina 10.15 的 Mac 上，打开 Finder 访达。在装有 macOS Mojave 10.14 或更低版本的 Mac 上或者 PC 上，打开 iTunes；
  2. 如果出现信息询问设备密码或让你「信任此电脑」，请按屏幕上显示的步骤操作；
  3. 确保已在电脑上找到你的设备（即已连接完成）；
  4. 如果你想存储设备或 Apple Watch 中的「健康」和「健身记录」数据，则需要对备份进行加密：选择名为「加密备份」的框并设置一个能够记住的密码[8]。如果不需要存储“健康”和“健身记录”数据，则可进行不加密的备份；
  5. 点按「立即备份」按钮；
  6. 等待备份过程结束

在通过 PC/Mac 恢复时，你需要先确定你在备份时所设置的密码（如果选择了「加密备份」）是否正确，然后按照以下步骤尝试进行恢复：

  1. 在装有 macOS Catalina 10.15 的 Mac 上，打开 Finder 访达。在装有 macOS Mojave 10.14 或更低版本的 Mac 上或者在 PC 上，打开 iTunes；
  2. 使用 USB 连接线将你的设备连接到电脑。如果出现信息询问设备密码或让您“信任此电脑”，请按屏幕上显示的步骤操作；
  3. 当您的 iPhone、iPad 或 iPod touch 出现在访达窗口或 iTunes 中时，请选择该设备；
  4. 选择「恢复备份」。如果有信息提示设备的软件版本太旧，请更新你的设备；
  5. 查看各个备份的日期，选择相关性最高的备份[9]；
  6. 点按「恢复」按钮，并等待恢复完成。如果系统提示输入加密备份的密码，请输入密码
  7. 等待恢复过程结束

至此，你的 iPhone 和 iPad 将恢复到先前的可用状态

macOS

备份 Mac 的最好方式是使用 macOS 内置的「时间机器」。你可以使用时间机器对你的所有文件进行自动备份，包括应用、音乐、照片、电子邮件、文稿和系统文件。如果你拥有备份，当原始文件从 Mac 永久性删除或者 Mac 中的硬盘（或 SSD）被抹掉或更换时，还可以从备份恢复文件

创建时间机器备份

要使用时间机器创建备份，你只需一个外置储存设备。连接设备并选择它作为你的备份磁盘后，时间机器就会自动制作过去 24 小时每小时的备份、过去一个月每天的备份以及之前所有月份每周的备份。如果备份磁盘已满，则最早的备份将会被删除[10]

使用时间机器须连接以下其中一种外置储存设备：

  • 连接到 Mac 的外置驱动器，例如 USB、Thunderbolt 雷雳或 FireWire 驱动器；
  • 连接到 AirPort Extreme 基站（802.11ac 机型）或 AirPort 时间返回舱的外置驱动器；
  • AirPort 时间返回舱；
  • 共享为「时间机器」备份目标位置的 Mac；
  • 支持通过 SMB 进行「时间机器」备份的 NAS 设备

将外置驱动器直接连接到 Mac 时，系统可能会询问你是不是要通过「时间机器」将这个驱动器用于备份。请选择「给备份磁盘加密」（推荐但不必须），此情况下只有知道密码的用户才能访问加密的备份。接着点按「用作备份磁盘」，然后你的外置驱动器将被抹掉，并使用时间机器支持的磁盘格式存储备份。此操作将抹掉你外置驱动器上的所有数据，无法恢复。要想让 Time Machine 备份不占用你的整个外置磁盘可用空间，你可以先对该外置驱动器进行「分区」操作

如果时间机器没有询问是不是使用你的驱动器，请按照以下步骤手动进行添加：

  1. 从菜单栏中的「时间机器」菜单中，打开「时间机器」偏好设置。或者选取「系统偏好设置」，然后点按「时间机器」；
  2. 点按「选择备份磁盘」（或「选择磁盘」或「添加或移除备份磁盘」）；
  3. 从可用磁盘列表中选择你的外置驱动器。然后选择「加密备份」（推荐但不必须）并点按「使用磁盘」；
  4. 如果你选择的磁盘没有按照「时间机器」的要求进行格式化，系统会提示你先抹掉磁盘。点按「抹掉」以继续操作。这样操作会抹掉备份磁盘上的所有信息

选择备份磁盘后，「时间机器」会立即开始自动定期备份，无需你进行进一步操作。首次备份可能需要较长时间，具体取决于文件的数量，但在备份期间你可以继续使用 Mac。「时间机器」只会备份自上次备份以来有变动的文件，因此将来的备份速度会加快

要手动启动备份，请从菜单栏的「时间机器」菜单中选取「立即备份」。使用这个菜单，还可以查看某一备份的状态或跳过正在进行的某一备份

恢复时间机器备份或恢复某一（些）文件

利用 macOS 恢复功能，你可以轻松重新安装 Mac 操作系统，即使你需要先抹掉启动磁盘也是如此。你只需连接到互联网即可。如果有可用的无线网络，你可以从 Wi-Fi 菜单中选取这个无线网络，也可在 macOS 恢复功能中找到无线网络

要通过 macOS 恢复功能启动，请将 Mac 开机，然后立即按住键盘上的以下组合键之一。当你看到 Apple 标志、旋转的地球或其他启动屏幕时，请松开这些按键：

  • Command - R：重新安装你在 Mac 上安装过的最新 macOS（建议）；
  • Option - Command - R：升级到与你的 Mac 兼容的最新 macOS；
  • Shift - Option - Command - R：重新安装 Mac 随附的 macOS 或与它最接近且仍在提供的版本

系统可能会提示您输入密码，例如固件密码，或这台 Mac 的管理员用户的密码。请输入要求的密码以继续。当你看到实用工具窗口时，即表示你已通过 macOS 恢复功能启动

在做好重新安装 macOS 的准备后，请从「实用工具」窗口中选取「重新安装 macOS」。然后点按「继续」，并按照屏幕上的说明进行操作。系统将提示你选取一个作为安装目标的磁盘

  • 如果安装器要求解锁你的磁盘，请输入你在登录 Mac 时所使用的密码；
  • 如果安装器无法识别你的磁盘，或者指出它无法安装到你的电脑或宗卷上，你可能需要先抹掉磁盘；
  • 如果安装器让你选择是安装在「Macintosh HD」上还是在「Macintosh HD-Data」上，请选取「Macintosh HD」

请不要将 Mac 置于睡眠状态或合上上盖，以便安装完成。在安装过程中，Mac 可能会多次重新启动并显示进度条，并且屏幕可能每次都处于空白状态并持续几分钟时间

而「时间机器」还可用于恢复 macOS 上的某一（些）文件。步骤也十分简单

如果你要恢复所有文件：

  1. 确保你的「时间机器」备份磁盘已连接到 Mac 并处于打开状态。然后将 Mac 开机；
  2. 打开「迁移助理」，它位于「应用程序」文件夹中的「实用工具」文件夹内；
  3. 当系统询问你希望以哪种方式转移信息时，请选择相应的选项，以便从 Mac、「时间机器」备份或启动磁盘进行转移。然后，点按「继续」；
  4. 选择你的「时间机器」备份，然后点按「继续」；
  5. 如果系统提示你从按日期和时间排列的备份列表中进行选择，请选择一个备份，然后点按「继续」；
  6. 选择要转移的信息，然后点按「继续」以开始转移。这个屏幕看起来可能与你 Mac 显示的屏幕有所不同；
  7. 如果内容很多，则转移可能需要几个小时才能完成。转移完成后，请重新启动你的 Mac，并登录迁移的帐户，以查看它的文件

如果你要恢复特定文件：

  1. 确保你的「时间机器」备份磁盘已连接并处于打开状态；
  2. 打开包含或曾经包含你要恢复的项目的窗口。例如，要恢复你从「文稿」文件夹中删除的文件，首先要打开「文稿」文件夹。或者，如果你使用的应用会在你处理文稿时自动存储文稿版本，请打开这个文稿的任意版本；
  3. 从「时间机器」菜单中选取「进入时间机器」；
  4. 使用屏幕边缘的时间线来查看“时间机器”备份中在相应日期和时间所备份的项目。时间线可能还包含本地快照[11]。使用屏幕上的向上和向下箭头跳至上次窗口内容变化的时间点。你也可以使用窗口中的搜索栏来查找项目，然后沿时间线移动并关注该项目的变化情况。选择一个项目并按下空格键来预览该项目，并确保它是您要找的文件；
  5. 点按「恢复」以恢复你所选的项目，或按住 Control 键点按这个项目以显示其它选项

至此，你的 Mac 将恢复到先前的可用状态

watchOS

Apple Watch 内容会自动备份到配对的 iPhone 上，因此你可以从 iPhone 备份中恢复 Apple Watch（当然，二者必须同时恢复）。当你将 iPhone 备份到 iCloud 或 PC/Mac 上时，你的 iPhone 备份还将包括你的 Apple Watch 数据

请注意一旦安装测试版本软件，你的 Apple Watch 便无法恢复到之前发布的操作系统版本。如果出现需要恢复先前状态的情况，一般也只有寻求 Apple 官方通过返厂这一种途径来完成恢复

你的 Apple Watch 数据备份将包括：

  • 特定于 App 的数据（适用于内建 App）和设置（适用于内建 App 和第三方 App）。例如，地图、距离、单位以及你的「邮件」、「日历」、「股市」和「天气」设置；
  • 主屏幕的 App 布局；
  • 表盘设置。包括你当前的表盘、自定设置和顺序
  • 「程序坞」设置，包括顺序（是按收藏的 App 还是按最近使用的 App 进行排序），以及可用的 App；
  • 通用系统设置。例如表盘、亮度、声音和触感设置；
  • 健康和健身数据（例如 Apple Watch 中的历史记录、成绩、「体能训练」和「健身记录」校准数据），以及用户输入的数据[12]；
  • 通知设置；
  • 同步到你的 Apple Watch 和你的「音乐」设置的播放列表、专辑和混合曲目；
  • Apple Watch Series 3 或更新机型的 Siri 语音反馈设置，用于控制 Siri 说话的时机；
  • 已同步的相簿[13]；
  • 时区

但其备份中将不包括：

  • 蓝牙配对
  • Apple Watch 上用于 Apple Pay 的信用卡或借记卡
  • Apple Watch 的密码

  11. 你的「时间机器」备份磁盘有时可能不可用，因此「时间机器」还会将某些备份储存在你的 Mac 上。这些备份称作本地快照 ↩︎

  12. 要备份健康和健身数据，你需要使用 iCloud 或加密的 iTunes 备份 ↩︎

  13. 要查看同步了哪个相簿，请打开 Apple Watch App，轻点「我的手表」标签，然后轻点「照片」>「已同步相簿」 ↩︎
