博客基础内容

目录功能

@[TOC]

标题

用 # 表示标题。标题嵌套则在下一级引用后跟一个 > 来表示，标题嵌套没有数量限制

  • 一级标题：#
  • 二级标题：##
  • 三级标题：###

文字修饰

  • 粗体：**粗体** 或 __粗体__
  • 斜体：*斜体* 或 _斜体_
  • 删除线：~~删除线~~
  • 斜粗体：***斜粗体*** 或 ___斜粗体___
  • 斜粗体删除线：~~***斜粗体删除线***~~
  • 文本高亮：==文本高亮==

代码块及代码高亮

代码块

  • 单行代码块：代码内容
  • 多行代码块：三个反引号，另起一行开始写代码，最后一行再加上三个反引号

代码高亮

三个反引号后面加上编程语言的名字，另起一行开始写代码，最后一行再加上三个反引号。形如：

  • Java
public static void main(String[]args){} //Java
  • C
int main(int argc, char *argv[]) //C
  • Bash
echo "hello GitHub" #Bash
  • JavaScript
document.getElementById("myH1").innerHTML="Welcome to my Homepage"; //javascipt
  • Diff
+ 鸟宿池边树，僧敲月下门
- 鸟宿池边树，僧推月下门

用 > 表示引用。引用嵌套则在下一级引用后跟一个 > 来表示，嵌套没有数量限制。形如：

列表

无序列表

用 - 或 + 或 * 后跟一个空格来表示无序列表，顺序没有特别要求。嵌套则在下一级列表前加上空格以确保下一级的列表符号在上一级列表的第一个文本字符的正下方，无序列表嵌套没有数量限制。形如：

  • 一级无序列表
    • 二级无序列表
      • 三级无序列表

顺序列表

用数字后跟一个空格来表示顺序列表。形如：

  1. 顺序列表
  2. 顺序列表
  3. 顺序列表

略缩信息

缩略信息紧跟在内容后面的一行，以 *[缩略词]:缩略内容 为格式。形如：

  • Github 是软件源代码托管服务平台

角标

使用 内容^上标内容^内容 即可表示上标，使用 内容~下标内容~内容 即可表示下标。形如：

  • 210=1024
  • H2O 是水的分子式

脚注

脚注可以直接使用 ^[脚注内容] 来表示，或者使用 ^[表示顺序的数字] 来表示。第二种方法需在下一行紧跟 ^[表示顺序的数字]:脚注内容 以输入脚注的具体内容。如果不想使用表示顺序的数字来表示，也可以使用 n 来代替数字，此时这个表示顺序的数字会被自动计算。脚注会显示在全文下方，并在脚注内容后显示一个返回脚注位置的按钮。形如：脚注示例[1]（该示例将显示在全文下方）

图片

  1. 使用 ![图片描述（可选）](图片链接) 就可以插入图片；
  2. 使用 <img src="图片链接"> 也可以插入图片。这并不是 Markdown 的语法，而是 HTML 语法；
  3. 链接和图片的 HTML 语法可混合使用;
  4. 在 标签中加入 loading="lazy" 属性可在受支持的浏览器中对图片进行懒加载

链接

  1. 使用 <链接地址> 就可以直接表示链接；
  2. 使用 [链接描述](链接地址) 也可以表示链接；
  3. 使用 <a href="链接地址">链接描述</a> 可以表示在新标签页打开链接。这并不是 Markdown 的语法，而是 HTML 语法。使用 <a href="链接地址" target="_blank">链接描述</a> 可以表示在新标签页打开链接；
  4. 链接和图片的 HTML 语法可混合使用

LaTeX

  1. 控制序列的概念（类似于函数）。控制序列可以是作为命令：以 \ 开头，参数：必须参数（{}）和可选参数（[]），最外侧用 $ 包裹
  2. 可以排版公式与文字，故分为：数学模式和文本模式。如果你想要在公式中排版普通的文本（直立字体和普通字距），那么你必须要把这些文本放在 \textrm{...} 命令中

简单显示

  • ∑i=0ni2\sum_{i=0}^{n}i^2∑i=0n​i2

居中显示

  • ∑i=0ni2\sum_{i=0}^{n}i^2 i=0∑n​i2

LaTeX 的角标

上标命令是 ^{角标}，下标命令是 _{角标}。当角标是单个字符时可以不用花括号（花括号是用于分组，即花括号内部文本为一组）。如果你想要把等式的一部分排版成罗马字体，不要用 \textrm 命令，因为 \textrm 会暂时切换到文本模式， 而此时字体大小切换机制将不起作用。使用 \mathrm 来保持字体大小切换机制的正常。形如：

  • x1x_1x1​
  • x12x_1^2x12​
  • x12x^2_1x12​
  • x22(n)x_{22}^{(n)}x22(n)​
  • ∗ ⁣x∗{}^*\!x^*∗x∗

空格

  • 紧贴：a ⁣ba\!bab
  • 没有空格：ababab
  • 小空格：a,b
  • 中等空格：a;b
  • 大空格：a\ b
  • 全空格：aba\quad bab
  • 两个全空格：aba\qquad bab

分式

输入较短的分式时，最简单的方法是使用斜线。譬如输入 (x+y)/2，可得到 (x+y)/2。要输入带有水平分数线的公式，可用命令：\frac{分子}{分母}。形如：

  • x+y2\frac{x+y}{2}2x+y​
  • 11+12\frac{1}{1+\frac{1}{2}}1+21​1​
  • α=β=arccos⁡r1+r2d\alpha=\beta=\arccos\frac{r1+r2}{d}α=β=arccosdr1+r2​

根式

排版根式的命令是：开平方：\sqrt{表达式}；开 n 次方：\sqrt[n]{表达式}。形如：

  • 2<33\sqrt{2}<\sqrt[3]{3}2​<33​
  • 1+1+a2p\sqrt{1+\sqrt[p]{1+a^2}}1+p1+a2​​
  • 1+1+a2p ⁣\sqrt{1+\sqrt[^p\!]{1+a^2}}1+p1+a2​​

求和与积分

排版求和符号与积分符号的命令分别为 \sum 和 \int，它们通常都有上下限，在排版上就是上标和下标。形如：

  • ∑k=1n1k\sum_{k=1}^{n}\frac{1}{k}∑k=1n​k1​
  • ∑k=1n1k\sum_{k=1}^n\frac{1}{k}∑k=1n​k1​
  • ∫abf(x)dx\int_a^b f(x)dx∫ab​f(x)dx
  • ∫abf(x)dx\int_a^b f(x)dx∫ab​f(x)dx
