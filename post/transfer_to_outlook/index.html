登录邮箱修改记

2020-06-08 · 7 min read · 0
Narrative

这是一篇发布于 5 年前的文章，其中的信息可能已经有所发展或是发生改变

信都收不到，我还要它干啥？

描述一下问题的背景

最近 Epic Games 不是每周送游戏大作嘛（不是吧？真有人领了还把它下载下来玩？🤣），之前也在 Epic Games 上领了很多免费游戏，但是从这次活动开始，领取游戏之前要先设置双因素验证了

于是……我等了将近一个晚上，愣是没收到任何来自 Epic Games 的验证邮件。找了垃圾邮箱，也找了 QQ 邮箱自己的收信和拦截记录，依旧没有。遂放弃，转而使用 Microsoft 的验证器应用开启双因素验证

其实我之前也遇到过 QQ 邮箱收不到信的问题，无论信是来自国内还是国外，但当时没怎么在意。这里说的是压根收不到信，而不是那种过了很长时间之后堵在路上的信一窝蜂全来了这种情况

之前还有过另一种情况，给别人发邮件，我邮件投出去几天了，愣是没收到回信。一直以为是对方时间所以没看或者什么其它的情况。结果后来联系对方，对方摊手说是压根就没收到，也并不在垃圾邮箱里……（而 QQ 邮箱发件记录居然还清楚地写着「投递成功」🙄️）

确定方案，开始人工智障般的修改

由于我一共拥有 3 个邮箱，一个 QQ 邮箱（即将修改的那个）、一个 Outlook（平时有的时候注册账户时会直接上这个邮箱）和一个 Gmail（好像也就 Setapp 和一些用 Google 登录的内容会用到这个邮箱，比如 Notion）。因此，我就直接选择了「使用 Outlook 代替 QQ 邮箱」的方案

无论我如何选择方案，我都只能一个一个手动的去修改绑定的邮箱地址，就如同一个智障一般……

修改邮箱地址时遇到的种种问题

虽然说修改的步骤如同智障一般，但在修改邮箱的时候也遇到了一些「奇奇怪怪」、让我充分开动脑筋解除修改过程中「乏味感」的各种问题

压根不给我修改的机会

这个问题嘛……还真是无法解决🤷‍♂️。不过好在这样的网站不多，我自己用的也不多

修改入口设置的奇奇怪怪，还不支持某些邮箱域名

要说这个问题，某国内最大的中文搜索引擎绝对是榜单第一名。那奇奇怪怪的修改入口就算了（害，没给个二维码让我下载手机 XX App 继续操作我就已经得谢谢它了），居然还不支持 Outlook……看了半天，除了 QQ 邮箱外，其它支持的几个邮箱都是奇奇怪怪的邮箱域名，查了一下，大多是国内的

我曾经碰到过注册时不允许我使用自己的阿里域名邮箱的，但是不支持 Outlook 这样的邮箱的也太奇怪了吧……

看了一遍，支持的都是国内的各种邮箱……

既然不支持，那就只好作罢。反正这个「国内最大的中文搜索引擎」也已经是被我边缘化的东西了

修改入口找到了，但是……死活收不到验证邮件

一部分网站的账号修改邮箱地址，是需要之前绑定的邮箱收取一个验证码验证本人操作的。但是……有的网站更改完邮箱地址后我一直收不到后续的确认邮件

可是让我想修改登录地址的就是因为这个原因啊！！！

在这个问题上我要点名那个由两位曾经模范恩爱的夫妻共同经营，现在没事就摔个杯子、夺个公章、发条微博互相爆料对方私生活的某电商网站。其实后来有了 Kindle 之后就不怎么在它家买书了，而且现在配送速度也已经远远不是刚刚有名气那个时候的配送速度了，可惜啊……

修改基本完成，但我仍然需要保有 QQ 邮箱

花了将近 1 个半小时的时间，可算是把这个机械化的项目完成的差不多了。将近 130 个网站、App 的绑定邮箱/登录邮箱全部从 QQ 邮箱转到了 Microsoft 的 Outlook。但因为有一些软件授权的关系，我仍然需要继续保有这个邮箱

修改之后，顺眼多了～

这年头，大家 QQ、WeChat 啥的 App 一大堆，但我仍然建议对于那些不重要的、不急的事情，用邮箱联系我。不用在意什么格式，只要你用的是中文或者英文，我都能看懂……

写在最后，回答几个问题……

Q：现在手机号登录这么好用，为什么不干脆用手机号登录呢？

A：不是吧？不是吧？你登录一次就要收一条手机验证码，不觉得这个设定很烦而且很没有效率吗？

很多国外的服务是不支持你用手机号注册登录的（例子太多，不举了）。我承认手机号码有很多优点，比如说不用再记密码了。但是，手机号登录这个方法真的不适合我。你能想象你每次手机号登录完，删短信的痛苦吗？？？

何况我手机删除一遍，Apple Watch 上还要再删除一遍！！！（🌶️🐔🍎💊）

Q：哪一个账号是你最先修改的？解释一下为什么

A：第一个修改的账号当然是我的 Apple ID。虽然是国区账号（话说国区账号好像一直都在一个隐形的「鄙视链」的最底层），但里面包含我几乎所有与 Apple 设备有关的内容（主要是很多已购买的项目）。至于为什么不直接转外区，实在不是不想转，而是本人实在是没什么条件能够解决转区后的支付问题，而直接抛弃国区账号好像有点「矫枉过正」，也实在是有点对不起我那干瘪的钱包……因此 Apple ID 是我最先修改的账号

由于我另外有一个香港区的 Apple ID（你懂的～），用的是 Outlook，所以国区的这个账号也就无法使用 Outlook 了。遂尝试使用 Gmail，试了一下发现也不行（好像是因为被我那个香港区的 Apple ID 作为备用救援邮箱啥的了，总之就是被占用了），所以只能使用当时注册 Apple ID 时同时设置的 iCloud 邮箱了（我其实一直觉得这个 iCloud 邮箱没什么用，当时注册纯粹是因为好奇。我也不清楚现在注册的 Apple ID 有没有还同时获得一个 iCloud 邮箱了，反正我注册香港区 Apple ID 的时候没有这个要求）

当然，这个方案也有一个缺点，修改为 iCloud 邮箱作为自己的 Apple ID 后，就再也无法变更自己的 Apple ID 地址了。据说是在 iOS 11 还是 12 的时候，新注册的 Apple ID 支持使用手机号注册（中国大陆特供）。我知道有这个功能，但我的账号是早先注册的旧账号，Apple 也不提供将 Apple ID 修改为手机号码这个功能

说来奇怪，自从我 Apple ID 变为 iCloud 邮箱后，以前那些每隔一段时间就会收到的什么澳门赌场、在线发牌之类的广告 iMessage 和日历项目就再也没收到过……
